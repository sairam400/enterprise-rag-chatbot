# Sample Documents

Four generated documents used by the eval harness and the quickstart demo,
one per supported file type:

- `hr_policies.md` — vacation, expense, and remote work policy
- `security_policy.txt` — password, MFA, and incident reporting policy
- `onboarding_guide.docx` — new hire onboarding steps (generated by
  [`evals/generate_sample_docs.py`](../evals/generate_sample_docs.py))
- `product_faq.pdf` — pricing, refunds, support SLA (also generated by the
  same script)

`evals/dataset.json` has 12 questions with known answers in these documents,
plus 2 questions nothing here answers, to check that the bot says so instead
of guessing.
